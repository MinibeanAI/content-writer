# 自媒体文章模板

## 1. 观点文结构

```
标题：引发共鸣/争议的观点
开头：引入现象+抛出观点
正文：
  - 观点1 + 案例/数据支撑
  - 观点2 + 案例/数据支撑
  - 观点3 + 案例/数据支撑
结尾：总结+互动引导（评论/转发）
```

## 2. 干货文结构

```
标题：XXX干货/指南/攻略
开头：痛点引入+承诺收益
正文：
  - 背景/前提条件
  - 方法/步骤1
  - 方法/步骤2
  - 方法/步骤3
  - 避坑指南/常见错误
结尾：总结+行动呼吁
```

## 3. 故事文结构

```
标题：冲突/悬念式
开头：场景引入
正文：
  - 背景介绍
  - 冲突出现
  - 转机/努力
  - 结果/感悟
结尾：升华+观点输出
```

## 4. 清单文结构

```
标题：XXX清单/必看/TOP10
开头：为什么需要这个清单
正文：
  - 项1 + 简短说明
  - 项2 + 简短说明
  - ...
结尾：总结+互动
```
