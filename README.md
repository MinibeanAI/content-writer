# content-writer

自媒体文章写作助手 - Skill

## 功能 (Features)

- 热门选题搜索 / Hot Topic Search
- 自动写文章 / Auto Article Writing
- 智能配图（支持7种风格）/ Smart Image Generation (7 styles)
- 自动排版 / Auto Formatting
- 多平台发布 / Multi-platform Publishing

## 使用方法 (How to Use)

1. 安装 Skill 到 OpenClaw / Install Skill to OpenClaw
2. 告诉它想写什么主题 / Tell it what topic you want to write about
3. 选择风格和排版 / Choose style and layout
4. 自动生成文章和配图 / Auto generate article and images

## 配图风格 (Image Styles)

- 手写手账 / Handwritten Journal
- 板报风格 / Blackboard Style
- 科技版式 / Tech Style
- 日常科普 / Flat Education
- 扁平简约 / Minimalist
- 孟菲斯 / Memphis
- 蒸汽波 / Vaporwave

## 图片生成 API (Image Generation APIs)

支持多种图片生成服务 / Supports multiple image generation services:

- 火山引擎 Seedream 5.0
- OpenAI DALL-E 3
- Google Gemini Image
- nano-banana
- Replicate
- Midjourney

用户可自行配置自己的 API Key。/ Users can configure their own API keys.

## 对话流程 (Dialogue Flow)

```
用户：写文章
助手：你想写什么主题？
用户：职场成长
助手：推荐3个选题，你选哪个？
用户：选第一个
助手：文章写好了，保存到哪里？
用户：本地
助手：需要配图吗？
用户：需要
助手：你想用哪种风格？是否需要看一下示例图？
用户：板报风格
助手：标题居中还是左对齐？配色用哪种？
用户：居中，紫色
助手：生成中...
助手：完成！文件已保存到 Downloads 文件夹
```
