# content-writer

自媒体文章写作助手 - OpenClaw Skill

## 功能

- 热门选题搜索
- 自动写文章
- 智能配图（支持7种风格）
- 自动排版
- 多平台发布

## 使用方法

1. 安装 Skill 到 OpenClaw
2. 告诉它想写什么主题
3. 选择风格和排版
4. 自动生成文章和配图

## 配图风格

- 手写手账
- 板报风格
- 科技版式
- 日常科普
- 扁平简约
- 孟菲斯
- 蒸汽波

## 图片生成 API

支持多种图片生成服务：
- 火山引擎 Seedream 5.0
- OpenAI DALL-E 3
- Google Gemini Image
- nano-banana
- Replicate
- Midjourney

用户可自行配置自己的 API Key。
